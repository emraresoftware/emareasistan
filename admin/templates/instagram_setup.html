Instagram Webhook Kurulum Rehberi

Meta Developer Console üzerinden Instagram Messaging API'yi adım adım bağlayın.

1. Meta Developer Console'a Gidin

Önce bir Meta (Facebook) geliştirici hesabı ve uygulama oluşturun.

Meta for Developers → Uygulamalar

2. Instagram Ürününü Ekleyin

  • Uygulamanızı seçin veya yeni oluşturun
  • Ürün Ekle → Instagram → Instagram Graph API veya Messenger Platform
  • Instagram Business veya Creator hesabınızı Facebook Page'e bağlayın

3. Webhook Ayarları

Instagram → Ayarlar → Webhooks bölümünde aşağıdaki bilgileri girin:

Verify Token'ı aynen kopyalayıp Meta'ya yapıştırın. Doğrulama butonuna bastığınızda Emare Asistan sunucusu bu token ile yanıt verecektir.

4. Abonelik Alanları

Webhook doğrulandıktan sonra şu alanlara abone olun:

  • messages – Gelen mesajlar
  • messaging_postbacks – Buton tıklamaları (opsiyonel)

5. .env Ayarları

Sunucunuzdaki .env dosyasına ekleyin:

# Instagram Messaging API
INSTAGRAM_PAGE_ID=
INSTAGRAM_ACCESS_TOKEN=your_page_access_token
INSTAGRAM_VERIFY_TOKEN=emare_verify

Access Token'ı Meta Developer Console → Instagram → Token Araçları'ndan alın. Page Access Token kullanın (Instagram Business hesabına bağlı Facebook Page).

Önemli Notlar

  • 24 saat kuralı: Müşteri mesaj gönderdikten sonra 24 saat içinde yanıt verebilirsiniz. Sonrasında template mesaj gerekir.
  • İzinler: instagram_basic, instagram_manage_messages, pages_manage_metadata gerekebilir.
  • Canlı mod: Test modunda sadece uygulama rollerindeki hesaplar mesaj alır. Canlıya geçince herkes kullanabilir.
