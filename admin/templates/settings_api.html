YENİ

Yazılımlarınızdaki hataları, kritik uyarıları ve sistem bildirimlerini anında WhatsApp'a gönderin. api_key aşağıdan oluşturun, alert_phonea bildirim alacak numarayı yazın.

Uyarıların gönderileceği WhatsApp numarası veya grup JID'i
POST /api/alert isteğinde gönderilecek gizli anahtar
📋 Entegrasyon Kodu Örnekleri (Python / JavaScript / curl)
Python (requests)
import requests

requests.post("{{ api_base }}/api/alert", json={
    "api_key": "{{ alert_api_key if alert_api_key and alert_api_key != '••••••••••••' else 'YOUR_API_KEY' }}",
    "level": "critical",      # critical | error | warning | info
    "app": "SatışPaneli",
    "title": "Ödeme servisi yanıt vermiyor",
    "message": "PaymentGateway.charge() TimeoutError (>30s)",
    "trace": traceback.format_exc(),  # opsiyonel
})
JavaScript / Node.js
await fetch("{{ api_base }}/api/alert", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    api_key: "YOUR_API_KEY",
    level: "error",
    app: "Web Sitesi",
    title: error.message,
    trace: error.stack,
  }),
});
